# Tfg-Excursiones-Lanzarote
Aplicación Web de venta de excursiones en Lanzarote, donde se mezcla naturaleza y arte. Desarrollada como Trabajo Final de Grado (DAW)
Tecnologías.

HTML, CSS, SQL,JavaScript, PHP, JSON, AJAX, MYSQL, VSC.

Funcionalidades

  La aplicación implementa las siguientes funcionalidades principales:

  Visualización de la página principal.
  Sistema de autenticación para usuarios y administradores.
  Gestión de reservas por parte del usuario:
  Añadir reservas
  Eliminar reservas
  Realizar compras
  Panel de administración con las siguientes capacidades:
  Gestión de reservas de usuarios
  Visualización de ventas de excursiones
  Gestión de excursiones
