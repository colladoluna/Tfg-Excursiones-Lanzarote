# Tfg-Excursiones-Lanzarote
Aplicación web desarrollada como Trabajo Final de Grado (DAW) orientada a la gestión de excursiones, permitiendo a los usuarios realizar reservas y a los administradores gestionar el sistema.

HTML, CSS, SQL,JavaScript, PHP, JSON, AJAX, MYSQL, VSC.

Funcionalidades

  La aplicación implementa las siguientes funcionalidades principales:

  Visualización de la página principal.
  Sistema de autenticación para usuarios y administradores.
  Gestión de reservas por parte del usuario:
    Añadir reservas
    Eliminar reservas
    Realizar compras
  Panel de administración con las siguientes capacidades:
    Gestión de reservas de usuarios
    Visualización de ventas de excursiones
    Gestión de excursiones
  
  El sistema cumple con los siguientes requisitos de calidad:

  Interfaz de usuario sencilla e intuitiva.
  Código estructurado y correctamente documentado.
  Rendimiento optimizado para una experiencia fluida.
